## 简介

这是一款开箱即用的**水墨化**组件库。

我们注意到有许多优秀的企业级UI设计语言和组件库。

作为炎黄子孙，我们十分推崇中国独有的美术风格：水墨。

我们整理出了一套具有中国传统风格的水墨风组件库。

希望这款中国传统风格的组件库，能用于一些小众的需求，能满足于一些对页面有美术追求，又希望能快速搭建的用户。

同时，我们希望能尽一分小小的力量，让更多人了解中国传统文化。
